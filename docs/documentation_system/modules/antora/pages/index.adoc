== Install Antora

=== Global installation

[source, bash]
....
npm i -g @antora/site-generator-default@3.0.0-alpha.1 \ //<.>
         @antora/cli@3.0.0-alpha.1 \ //<.>
         asciidoctor-kroki  //<.>
....
<.> Antora default site generator
<.> Antora command line tool
<.> Asciidoctor Kroki module to transform diagram texts into images
