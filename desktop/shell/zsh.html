Zsh and Oh My Zsh

Plugins

Most plugins I use provide autocompletion for commands and aliases for commonly used arguments.

In .zshrc
plugins=(
    git
    git-flow
    jsontools
    colorize (1)
    dircycle (2)
    z (3)
    ant
    nvm
    ng
    npm
    kubectl
    urltools (4)
    auto-notify
    zsh-autosuggestions
    zsh-syntax-highlighting (5)
)
1 Adds syntax colouring and output via ccat and cless.
2 Adds z abc to switch to a previously used directory using a couple of characters of its name.
3 Lets you cycle through recent directories with ctrl+shift+left/right
4 Adds urlencode and urldecode commands.
5 Adds syntax highlighting while typing commands.

Aliases

I define some aliases to override default behaviour, and add sudo to commands I often want to start from my regular user account.

In aliases.zsh
# confirm overwrites
alias rm='rm -I --preserve-root' (1)
alias mv='mv -i'
alias cp='cp -i'
alias ln='ln -i'

if [ $UID -ne 0 ]; then (2)
  alias apt='sudo apt'
  alias apt-up='sudo apt update && sudo apt upgrade'
  alias rsnapshot='sudo rsnapshot'
  alias dpkg='sudo dpkg'
fi
1 Do not delete /; prompt if deleting more than 3 files at a time.
2 Add sudo to these commands if not logged in as admin.
